# Stack Implementation in C++

## Overview

This project demonstrates the implementation of a Stack data structure using C++. The stack follows the Last In, First Out (LIFO) principle, where the last element added is the first element removed.

## Features

* Push operation (insert element)
* Pop operation (remove element)
* Peek/Top operation (view top element)
* Check if stack is empty
* Display stack elements
* User-friendly menu-driven interface

## Technologies Used

* C++
* Object-Oriented Programming (OOP)

## How to Run

1. Download or clone the repository.
2. Open the project in any C++ IDE or compiler.
3. Compile and run the source file.
4. Follow the menu options to perform stack operations.

## Learning Outcomes

* Understanding of Stack data structure
* Implementation of LIFO principle
* Practice with arrays/classes in C++
* Improved problem-solving and programming skills
